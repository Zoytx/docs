---
title: 'Get Workflow call logs'
api: 'GET workflows/logs/{workflow_id}'
description: 'Retrieve a list of all workflow logs.'
---

### Headers

<ParamField header="Authorization" type="string" required>
  Your API key for authentication. Prefix with `Bearer `.
</ParamField>

<ParamField header="x-access-token" type="string" required>
  Your access token for authentication.
</ParamField>

### Path Parameters

<ParamField path="workflow_id" type="string" required>
  The unique identifier of the workflow you want to get the logs of.
</ParamField>

### Response

<ResponseField name="data" type="array">
    An array containing the value of all the workflows logs as values.
</ResponseField>

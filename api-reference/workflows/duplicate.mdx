---
title: "Duplicate a Workflow"
api: "POST workflows/duplicate/{id}"
description: "Initiate the duplication of an existing workflow"
---

### Headers

<ParamField header="Authorization" type="string" required>
  Your API key for authentication. Prefix with `Bearer `.
</ParamField>

<ParamField header="x-access-token" type="string" required>
  Your access token for authentication.
</ParamField>

### Body

<ParamField body="id" type="string" required>
  The id of the existing workflow to duplicate from.
</ParamField>

<ParamField body="name" type="string" required>
  The name of the duplicated workflow
</ParamField>

<ParamField body="description" type="string" required>
  The description of the duplicated workflow
</ParamField>

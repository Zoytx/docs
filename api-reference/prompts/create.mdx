---
title: 'Create Prompt'
api: 'POST /v1/prompts'
description: Create and store a prompt for future use.
---

### Headers

<ParamField header="authorization" type="string" required={true}>
  Your API key for authentication.
</ParamField>

### Body

<ParamField body="prompt" type="string" required={true}>
  Prompt to store.
</ParamField>

<ParamField body="name" type="string">
  Name of prompt you want to store as reference.
</ParamField>

### Response

<ResponseField>
    Prompt object.
</ResponseField>
<ResponseField>
    Prompt to store.
</ResponseField>
<ResponseField>
    Name of prompt you want to store as reference.
</ResponseField>


<ResponseExample>
```json Response
{
  "status": "success",
  "prompt": {
    "prompt": "# Final Test",
    "name": "demo",
    "id": "PT-02b2ecdi-39f2-443f-8cb0-a6c854c65fc0"
  }
}


```
</ResponseExample>

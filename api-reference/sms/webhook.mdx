---
title: 'Update SMS Webhook'
api: 'POST /v1/sms/webhook/update'
description: Update the webhook for a given phone number.
---

### Headers

<ParamField header="authorization" type="string" required={true}>
  Your API key for authentication.
</ParamField>


### Body
<ParamField body="phone_number" type="string">
The phone number to update.
</ParamField>

<ParamField body="webhook" type="boolean">
The webhook to fire when an SMS is received.
</ParamField>


<ResponseExample>
```json Response
{
  "status": "success",
  "pathway_id": "9d404c1b-6a23-4426-953a-a52c392ff8f1"
}

```
</ResponseExample>
